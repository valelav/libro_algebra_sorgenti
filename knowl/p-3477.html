Capoverso

Sia \(A\) una matrice quadrata di ordine \(n\) a coefficienti in un campo \(K\text{.}\) Esistono allora polinomi irriducibili monici \(p_{1}\text{,}\) \(p_{2}\text{,}\) \(\ldots\text{,}\) \(p_{t}\) (non necessariamente distinti) e interi positivi \(\alpha_{1}\text{,}\) \(\alpha_{2}\text{,}\) \(\ldots\text{,}\) \(\alpha_{t}\) tali che \(A\) è simile alla matrice diagonale a blocchi
\begin{equation*} C\coloneqq \begin{pmatrix} C_{1} \amp 0 \amp \dots \amp 0 \\ 0 \amp C_{2} \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp C_{t} \end{pmatrix} \end{equation*}
dove ciascun \(C_{i}\) è la matrice compagna di \(p_{i}^{\alpha_{i}}\text{.}\) Inoltre, il polinomio caratteristico di \(A\) è \(p_{1}^{\alpha_{1}}\cdots p_{t}^{\alpha_{t}}\text{.}\) La matrice \(C\) è univocamente determinata a meno dell'ordinamento dei blocchi ed è detta forma canonica primaria di \(A\text{.}\)
