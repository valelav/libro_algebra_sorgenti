Capoverso

Sia \(A\) una matrice quadrata di ordine \(n\) a coefficienti in un campo \(K\text{.}\) Allora esistono polinomi monici \(f_{1}\mid f_{2}\mid\dots\mid f_{s}\) tali che \(A\) è simile alla matrice diagonale a blocchi
\begin{equation*} C\coloneqq \begin{pmatrix} C_{1} \amp 0 \amp \dots \amp 0 \\ 0 \amp C_{2} \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp C_{s} \end{pmatrix} \end{equation*}
dove ciascun \(C_{i}\) è la matrice compagna di \(f_{i}\text{.}\) Inoltre il polinomio caratteristico di \(A\) è \(f_{1}\cdots f_{s}\text{.}\) La matrice \(C\) è univocamente determinata ed è detta forma canonica razionale di \(A\text{.}\)
