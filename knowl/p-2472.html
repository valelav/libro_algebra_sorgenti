Capoverso

Si consideri l'anello \(\mat{3}{R}\) delle matrici di ordine \(3\) a coefficienti in un anello \(R\text{.}\) Prendiamo i sottoanelli \(S\) delle matrici del tipo
\begin{equation*} \begin{pmatrix} a_{11} \amp 0 \amp 0 \\ 0 \amp a_{22} \amp 0 \\ a_{31} \amp 0 \amp a_{33} \end{pmatrix} \end{equation*}
e il sottoanello \(T\) delle matrici triangolari superiori (per la verifica che \(S\) sia un sottoanello si veda l'Esercizio 17.3.5). La matrice \(A\coloneqq \begin{psmallmatrix} 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 0 \end{psmallmatrix}\) appartiene a \(S\) e, quindi, a \(S+T\text{,}\) mentre la matrice \(B\coloneqq \begin{psmallmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{psmallmatrix}\) appartiene a \(T\) e, quindi, a \(S+T\text{.}\) Il prodotto \(AB= \begin{psmallmatrix} 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \end{psmallmatrix}\) non appartiene a \(S+T\) (infatti in posizione \((2,3)\) sia le matrici di \(S\) che le matrici di \(T\) hanno valore nullo e così pure, quindi, le loro somme). Di conseguenza \(S+T\) non è un sottoanello di \(R\text{.}\)
