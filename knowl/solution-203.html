Soluzione 27.5.2.1.

Poiché il polinomio caratteristico ha grado \(6\text{,}\) le matrici cercate hanno ordine \(6\text{.}\) Tutti i blocchi di Jordan sono relativi all'unico autovalore \(-4\text{.}\) Poiché il polinomio minimo è \((x+4)^{3}\text{,}\) nella forma canonica di Jordan deve esserci un blocco di ordine \(3\) e altri blocchi di ordine minore o uguale a \(3\text{.}\) Considerando che l'ordine complessivo dei blocchi deve essere \(6\) possiamo avere \(2\) blocchi di ordine \(3\)
\begin{equation*} \begin{pmatrix} -4 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp -4 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp -4 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -4 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp -4 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -4 \end{pmatrix} \end{equation*}
oppure un blocco di ordine \(3\text{,}\) uno di ordine \(2\) e uno di ordine \(1\)
\begin{equation*} \begin{pmatrix} -4 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp -4 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp -4 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -4 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp -4 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -4 \end{pmatrix} \end{equation*}
oppure un blocco di ordine \(3\) e \(3\) di ordine \(1\)
\begin{equation*} \begin{pmatrix} -4 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp -4 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp -4 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -4 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp -4 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -4 \end{pmatrix}\text{.} \end{equation*}
