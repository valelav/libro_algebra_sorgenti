Capoverso
Il polinomio caratteristico di \(A\) può essere calcolato facilmente: si ha \(\det(x\Idmatrix-A)=(x-2)^{6}\text{.}\) Il polinomio caratteristico è totalmente riducibile e la matrice \(A\) ha come unico autovalore \(2\) di molteplicità algebrica \(6\text{.}\) Dunque la matrice è simile a una matrice di Jordan in cui avremo blocchi di Jordan di ordine complessivo \(6\text{.}\) Calcoliamo la successione dei ranghi associati all'autovalore \(6\) partendo, come sempre, dall'ordine della matrice che in questo caso è \(6\text{.}\) Consideriamo ora il rango della matrice \(A-2\Idmatrix\text{,}\) vale a dire la matrice
\begin{equation*} \begin{pmatrix} 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp 1 \amp 2 \amp 0 \\ 0 \amp 0 \amp 0 \amp 2 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 0 \end{pmatrix}\text{.} \end{equation*}
Si vede facilmente che il rango di questa matrice è \(4\text{.}\) Pertanto abbiamo \(6-4=2\) blocchi di Jordan relativi a \(2\text{.}\)
