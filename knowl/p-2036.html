Capoverso

Se \(\sim\) è una congruenza in un insieme \(S\) dotato di un'operazione, allora nell'insieme quoziente \(S/\sim\) è possibile definire un'operazione ponendo
\begin{equation*} \class[\sim]{x}\class[\sim]{y}\coloneqq\class[\sim]{xy}\text{.} \end{equation*}
Inoltre
  1. se l'operazione in \(S\) è associativa, allora l'operazione in \(S/\sim\) è associativa;
  2. se l'operazione in \(S\) è commutativa, allora l'operazione in \(S/\sim\) è commutativa;
  3. se l'operazione in \(S\) è dotata di elemento neutro \(1_{S}\text{,}\) allora \(\class[\sim]{1_{S}}\) è elemento neutro in \(S/\sim\text{;}\)
  4. se \(S\) è un monoide e \(x\) è un elemento invertibile in \(S\text{,}\) di inverso \(x^{-1}\text{,}\) allora \(\class[\sim]{x}\) è un elemento invertibile di inverso \(\class[\sim]{x^{-1}}\text{.}\)
