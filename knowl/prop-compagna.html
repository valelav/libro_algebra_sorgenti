Proposizione 27.3.2.

Sia \(V\) uno spazio vettoriale di dimensione finita \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{.}\) Supponiamo che \(V\text{,}\) con la struttura di \(K[x]\)-modulo indotta da \(\theta\text{,}\) sia ciclico di generatore \(\vect{v}\text{.}\) Se \(p\coloneqq a_{0}+a_{1}x+\dots+x^{n}\) è il polinomio monico che genera l'annullatore di \(\vect{v}\) allora \(\theta\text{,}\) rispetto alla base di \(V\) formata dai vettori \(\vect{v}_{1}\coloneqq\vect{v}\text{,}\) \(\vect{v}_{2}\coloneqq\vect{v}\theta\text{,}\) \(\dots\text{,}\) \(\vect{v}_{n}\coloneqq\vect{v}\theta^{n-1}\text{,}\) si rappresenta con la matrice
\begin{equation*} C(p)\coloneqq \begin{pmatrix} 0 \amp 1 \amp 0 \amp \dots \amp 0\\ 0 \amp 0 \amp 1 \amp \dots \amp 0\\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp 0 \amp \dots \amp 1\\ -a_{0} \amp -a_{1} \amp -a_{2} \amp \dots \amp -a_{n-1} \end{pmatrix} \end{equation*}
detta matrice compagna del polinomio monico \(p\text{.}\) Il polinomio caratteristico \(\det(x\Idmatrix-C(p))\) della matrice \(C(p)\) è esattamente \(p\text{.}\)
