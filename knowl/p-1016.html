Capoverso
È sottogruppo di \((\mat{n}{R},+)\) ma non è sottomonoide di \((\mat{n}{R},\cdot)\) (a parte il caso banale in cui \(n=1\)). Per \(n=2\) ecco due matrici simmetriche il cui prodotto non è una matrice simmetrica:
\begin{equation*} \begin{pmatrix} 1 \amp 0 \\ 0 \amp 0 \end{pmatrix} \begin{pmatrix} 0 \amp 1 \\ 1 \amp 0 \end{pmatrix}= \begin{pmatrix} 0 \amp 1 \\ 0 \amp 0 \end{pmatrix}\text{.} \end{equation*}
Questo esempio si può facilmente estendere a \(n\ge 2\text{.}\)
