Dimostrazione

Mostriamo che gli \(n\) vettori \(\vect{v}_{1}\text{,}\) \(\dots\text{,}\) \(\vect{v}_{n}\) sono linearmente indipendenti e sono quindi una base per \(V\text{,}\) che sappiamo avere dimensione uguale al grado del polinomio ordine di \(\vect{v}\text{.}\) Siano allora \(b_{1}\text{,}\) \(\dots\text{,}\) \(b_{n}\) elementi di \(K\) tali che \(\vect{v}_{1}b_{1}+\dots+\vect{v}_{n}b_{n}=\vect{0}\text{,}\) vale a dire \(\vect{v}b_{1}+\vect{v}(\theta-\lambda\Id)b_{2}+\dots+\vect{v}(\theta-\lambda\Id)^{n-1}b_{n}=\vect{0}\text{,}\) cioè \(\vect{v}(b_{1}+b_{2}(x-\lambda)+\dots+b_{n}(x-\lambda)^{n-1})=\vect{0}\text{.}\) Poiché l'annullatore di \(\vect{v}\) è generato da un polinomio di grado \(n\) ciò comporta che \(b_{1}+b_{2}(x-\lambda)+\dots+b_{n}(x-\lambda)^{n-1}\) è uguale a \(0\text{.}\) Se espandiamo questo polinomio notiamo che il coefficiente di \(x^{n-1}\) è \(b_{n}\) e, dunque, \(b_{n}=0\text{.}\) Una volta posto \(b_{n}=0\text{,}\) il coefficiente di \(x_{n-2}\) è \(b_{n-1}\) e, dunque, \(b_{n-1}=0\) e così via.
Mostriamo ora che la matrice rappresentativa di \(\theta\) rispetto a questa base è la matrice \(J_{n}(\lambda)\) data in enunciato. Per \(1\le j\le n-1\) si ha \(\vect{v}_{j}(\theta-\lambda\Id)=\vect{v}_{j+1}\) e, pertanto, \(\vect{v}_{j}\theta=\vect{v}_{j}\lambda+\vect{v}_{j+1}\text{.}\) Ciò permette di determinare i coefficienti della riga \(j\)-esima. Per determinare l'ultima riga della matrice notiamo che \(\vect{v}_{n}(\theta-\lambda\Id)=\vect{v}(\theta-\lambda\Id)^{n-1}(\theta-\lambda\Id)= \vect{v}(\theta-\lambda\Id)^{n}=\vect{0}\text{.}\) Pertanto \(\vect{v}_{n}\theta=\vect{v}_{n}\lambda\) da cui ricaviamo i coefficienti dell'ultima riga della matrice.
La matrice \(J_{n}(\lambda)\) è triangolare, quindi si calcola immediatamente il suo polinomio caratteristico \((x-\lambda)^{n}\) e, pertanto, l'unico autovalore è \(\lambda\) di molteplicità algebrica \(n\text{.}\) La molteplicità geometrica di \(\lambda\) è uguale a \(n-\rk(\lambda\Idmatrix-J_{n}(\lambda))\text{.}\) Ora
\begin{equation*} \lambda\Idmatrix-J_{n}(\lambda)= \begin{pmatrix} 0 \amp -1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp 0 \amp -1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp \dots \amp \dots \amp 0 \amp -1 \amp 0 \\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp -1\\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0 \end{pmatrix} \end{equation*}
ha, ovviamente, rango \(n-1\) e, dunque, la molteplicità geometrica di \(\lambda\) è uguale a \(1\text{.}\)
