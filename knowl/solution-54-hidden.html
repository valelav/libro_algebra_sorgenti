Abbiamo \(16\) matrici del primo tipo e \(16\) matrici del secondo tipo. In particolare, \(H\) è finito e, per la Proposizione 8.2.5, basta verificare che \(H\) è chiuso rispetto al prodotto.
Consideriamo tutti i possibili prodotti di matrici di \(H\text{.}\)
\begin{align*} \begin{pmatrix} \epsilon_{1} \amp 0 \\ 0 \amp \epsilon_{2} \end{pmatrix} \begin{pmatrix} \epsilon_{3} \amp 0 \\ 0 \amp \epsilon_{4} \end{pmatrix} \amp = \begin{pmatrix} \epsilon_{1}\epsilon_{3} \amp 0 \\ 0 \amp \epsilon_{2}\epsilon_{4} \end{pmatrix}\\ \begin{pmatrix} \epsilon_{1} \amp 0 \\ 0 \amp \epsilon_{2} \end{pmatrix} \begin{pmatrix} 0 \amp \epsilon_{3} \\ \epsilon_{4} \amp 0 \end{pmatrix} \amp =\begin{pmatrix} 0 \amp \epsilon_{1}\epsilon_{3} \\ \epsilon_{2}\epsilon_{4} \amp 0 \end{pmatrix}\\ \begin{pmatrix} 0 \amp \epsilon_{1} \\ \epsilon_{2} \amp 0 \end{pmatrix} \begin{pmatrix} \epsilon_{3} \amp 0 \\ 0 \amp \epsilon_{4} \end{pmatrix} \amp =\begin{pmatrix} 0 \amp \epsilon_{1}\epsilon_{4} \\ \epsilon_{2}\epsilon_{3} \amp 0 \end{pmatrix}\\ \begin{pmatrix} 0 \amp \epsilon_{1} \\ \epsilon_{2} \amp 0 \end{pmatrix} \begin{pmatrix} 0 \amp \epsilon_{3} \\ \epsilon_{4} \amp 0 \end{pmatrix} \amp =\begin{pmatrix} \epsilon_{1}\epsilon_{4} \amp 0 \\ 0 \amp \epsilon_{2}\epsilon_{3} \end{pmatrix} \end{align*}
I prodotti sono tutti elementi di \(H\) (i coefficienti che appaiono sono prodotti di elementi di \(U\) e sono quindi elementi di \(U\)).
