Proposizione 27.4.1.

Sia \(V\) uno spazio vettoriale di dimensione finita \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{.}\) Supponiamo che \(V\text{,}\) con la struttura di \(K[x]\)-modulo indotta da \(\theta\text{,}\) sia ciclico di generatore \(\vect{v}\) e che l'annullatore di \(\vect{v}\) sia generato dal polinomio \((x-\lambda)^{n}\text{.}\) I vettori \(\vect{v}_{1}\coloneqq \vect{v}\text{,}\) \(\vect{v}_{2}\coloneqq \vect{v}(\theta-\lambda\Id)\text{,}\) \(\dots\text{,}\) \(\vect{v}_{n}\coloneqq\vect{v}(\theta-\lambda\Id)^{n-1}\) formano allora una base di \(V\) rispetto a cui \(\theta\) si rappresenta con la matrice
\begin{equation*} J_{n}(\lambda)\coloneqq \begin{pmatrix} \lambda \amp 1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp \lambda \amp 1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp \dots \amp \dots \amp \lambda \amp 1 \amp 0 \\ 0 \amp \dots \amp \dots \amp 0 \amp \lambda \amp 1\\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp \lambda \end{pmatrix} \end{equation*}
detta blocco di Jordan 1  di ordine \(n\) relativo all'autovalore \(\lambda\text{.}\) Il polinomio caratteristico \(\det(x\Idmatrix-J_{n}(\lambda))\) della matrice \(J_{n}(\lambda)\) è esattamente \((x-\lambda)^{n}\text{.}\) L'endomorfismo \(\theta\) ha come unico autovalore \(\lambda\) di molteplicità algebrica \(n\) e molteplicità geometrica \(1\text{.}\)
