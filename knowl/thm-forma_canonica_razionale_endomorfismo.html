Teorema 27.3.7.

Sia \(V\) uno spazio vettoriale di dimensione \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{.}\) Esistono allora polinomi monici \(f_{1}\mid f_{2}\mid\dots\mid f_{s}\) e una base di \(V\) rispetto a cui \(\theta\) si rappresenta con la matrice diagonale a blocchi
\begin{equation*} C\coloneqq \begin{pmatrix} C_{1} \amp 0 \amp \dots \amp 0 \\ 0 \amp C_{2} \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp C_{s} \end{pmatrix} \end{equation*}
dove ciascun \(C_{i}\) è la matrice compagna del polinomio \(f_{i}\text{.}\) La matrice \(C\) è univocamente determinata.
