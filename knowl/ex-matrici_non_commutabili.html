Esempio 6.2.5.

Consideriamo l'operazione di moltiplicazione in \(\mat{2}{R}\) a coefficienti in un anello \(R\text{.}\) Prendiamo le matrici
\begin{equation*} A\coloneqq \begin{pmatrix} 0 \amp 1 \\ 0 \amp 0 \end{pmatrix} \qquad B\coloneqq \begin{pmatrix} 0 \amp 0 \\ 0 \amp 1 \end{pmatrix} \end{equation*}
e calcoliamo i prodotti \(AB\) e \(BA\text{:}\)
\begin{equation*} AB= \begin{pmatrix} 0 \amp 1 \\ 0 \amp 0 \end{pmatrix} \qquad BA= \begin{pmatrix} 0 \amp 0 \\ 0 \amp 0 \end{pmatrix} \end{equation*}
Poiché \(AB\ne BA\text{,}\) l'operazione di moltiplicazione riga per colonna non è commutativa.
